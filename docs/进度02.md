## 运行方式

```bash
cd web
python start_mast3r.py   # 启动服务器 (自动加载 MASt3R 模型)
```

## 访问

- 本地: http://localhost:8080
- 外网: cloudflared tunnel (自动分配 trycloudflare.com 域名)

## 依赖

- Python 3.11 (slam3r conda 环境)
- MASt3R-SLAM (thirdparty/mast3r + 本地编译 curope CUDA 内核)
- FastAPI + uvicorn
